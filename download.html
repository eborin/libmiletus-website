Downloading LibMiletus

You may download the latest ZIP version of LibMiletus from: https://github.com/MotorolaMobilityLLC/LibMiletus/zipball/master/
The ZIP version can be used to install LibMiletus on the Arduino IDE (See Getting Started).

You may also clone the whole repository using: 

git clone https://github.com/MotorolaMobilityLLC/LibMiletus.git

Downloading the MiletusApp

You may download a copy of the MiletusApp application, already compiled for Android, from one of the following links:

  • MiletusApp-1.1.0.apk
  • MiletusApp-1.0.0.apk

You may also clone the whole MiletusApp repository using: 

git clone https://github.com/MotorolaMobilityLLC/MiletusApp.git
